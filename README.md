Projeto simples desenvolvido para praticar HTML, JavaScript e Bootstrap 5.

 Tecnologias
- HTML5
- JavaScript
- Bootstrap 5

 Funcionalidades
- Inserção de dois números
- Cálculo automático da soma
- Validação de campos
- Interface responsiva

Objetivo
Praticar manipulação de DOM e lógica de programação.
